# SWAIF Factory Memory

This directory stores persistent project memory used by factory workflows.

- Owner: Delivery Lead (content), Platform/DevEx (automation behavior).
- Expected inputs: approved artifacts, decision outcomes, risk updates, and operational learnings.

## Files

- `project-context.md`: stable context for problem domain, constraints, and stakeholders.
- `decision-log.md`: chronological architectural/product/security decisions.
- `risk-register.md`: tracked risks, mitigations, status, and owners.

Keep entries concise, dated, and linked to spec/plan/task IDs whenever possible.
